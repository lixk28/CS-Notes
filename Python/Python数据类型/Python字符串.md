# 字符串的表示

Python不支持字符类型，单个字符就是长度为1的字符串。

Python中用成对的单引号`''`或双引号`""`中的内容表示字符串，可以使用成对的连续的三个引号`'''`或`"""`指定一个多行字符串。

```python
str1 = 'Hello, Python!'
str2 = "I am Groot."
str3 = '''Hello, Python!
 I am Groot'''
str4 = """Hello, Python!
 I am Groot"""

```

Python字符串的内容**可以是任意的**，因为是以Unicode编码的。
<img src=image/2021-02-13_21-00.png>

Python同时支持双引号和单引号，但是如果字符串中包含单引号或双引号，会出现问题：
<img src=image/2021-02-13_21-04.png>
Python会将第一个遇到的引号与第二个遇到的引号匹配，认为它们之间的内容是字符串。因此同时支持单引号和双引号可以部分解决这个问题，即字符串中只存在单引号或双引号其中一种。

但如果字符串中既有单引号，又有双引号，我们可以使用`\`对引号进行转义：
<img src=image/2021-02-13_21-09.png>
这样Python就会认为引号是字符串的一部分，而不是字符串的限定符。

## 字符串索引

在Python中，字符串中字符的索引有两个，分别是**从前向后的索引**，和**从后向前的索引**。

例如，字符串`Python`的索引：
```
front->end:  0   1   2   3   4   5
             P   y   t   h   o   n
front<-end: -6  -5  -4  -3  -2  -1
```

字符串访问运算符`[]`的语法格式：
```python
s[index]
```
它的功能是访问字符串给定索引处的字符。

## 字符串切片

字符串切片的语法格式为：
```python
string[<begin index>:<end index>]
或者
string[<begin index>:<end index>:<step length>]
```
其中，begin index是起始索引，end index是终止索引，step length是步长。

字符串切片的功能是提取字符串的一部分。
- 当不指定步长时，默认步长为1，会获取从起始索引到终止索引的子字符串，**注意这时切片得到的字符串包括起始索引处的字符，但不包括终止索引的字符**。
  - 起始索引和终止索引都可以被省略，起始索引被省略时表示切片始于开头，终止索引被省略时结束于末尾
  - 起始索引和终止索引可以超出字符串索引的范围，这样写的效果和被省略时相同
  - 若终止索引指定的元素位于起始索引的**前面或者相等**，切片的结果将会是空序列
  
  <img src=image/2021-02-15_14-03.png>

- 当指定步长时，从`起始索引`开始，不断获取索引处的字符，然后**令索引值加上步长（无论步长的正负）**，一直进行这个过程，直到索引值超出终止索引
    - 步长为正：从子字符串的前面向后面提取
    - 步长为负：从子字符串的后面向前面提取
    - 步长为0：错误，步长不可为0
    - :zap:**注意到**：步长为正时，要求`起始索引<终止索引`，否则将会得到空序列；步长为负时，要求`起始索引>终止索引`，否则将会得到空序列。但是当省略起始索引或终止索引时，字符将会被正确提取。

  <img src=image/2021-02-15_14-03.png>

# 字符串的格式化

## C风格的格式化字符串

Python支持格式化字符串的输出，提供了类似于C语言中`printf`函数类似的格式——`%`。

Python中，指定格式化字符串的值时，可以使用单个的值，元组，字典。其中最常见的是元组。

<img src=image/2021-02-15_17-41.png>

Python字符串格式化符号，与C基本完全一致：
| 符号       | 功能                               |
| ---------- | ---------------------------------- |
| `%c`       | 格式化字符及ASCII码                |
| `%s`       | 格式化字符串                       |
| `%d`       | 格式化整数                         |
| `%u`       | 格式化无符号整数                   |
| `%o`       | 格式化无符号八进制数               |
| `%x`       | 格式化无符号十六进制数             |
| `%X`       | 格式化无符号十六进制数（大写）     |
| `%f`       | 格式化浮点数，可指定小数点后的精度 |
| `%e`或`%E` | 用科学计数法格式化浮点数           |
| `%p`       | 十六进制格式化变量的地址           |

格式化操作符辅助指令：
| 符号    | 功能                                              |
| ------- | ------------------------------------------------- |
| `*`     | 定义宽度或小数点精度                              |
| `-`     | 左对齐                                            |
| `+`     | 在正数前显示正号                                  |
| `<sp>`  | 在正数前显示空格                                  |
| `#`     | 在八进制数前显示`0`，在十六进制数前显示`0x`或`0X` |
| `0`     | 显示的数字前填充`0`而不是默认的空格               |
| `%`     | 输出`%`                                           |
| `(var)` | 映射变量（字典参数）                              |
| `m.n.`  | m是显示的最小总宽度，n是小数点后的位数（可选）    |

## 字符串format方法

字符串的format方法可以格式化字符串。
字符串中用`{}`括起来的字段称为**替换字段**，format中指定的信息将被插入到字符串中，用来替换指定的字段。
如果要在结果中包含`{}`，可在格式字符串中用两个花括号来指定。

替换字段由下面三个部分组成，其中的每个部分都是可选的：
- **字段名**：索引或标识符，指出要设置哪个值的格式并使用结果来替换该字符串。
- **转换标志**：跟在感叹号后面的单个字符。当前支持的字符包括`r`（表示repr），`s`（表示str），`a`（表示ascii）。
- **格式说明符**：跟在冒号后面的表达式。可以详细地指定最终的格式，包括格式类型，字段宽度，数的精度，对齐方式，填充方式等。

### 替换字段名

一般情况下，需要指定替换字段名，然后在format中指定对应字段名的参数，这些参数将会被用于对应的替换字段。

你还可以不指定字段名，这样会按顺序将字段和参数匹配，这种方法实际上是自动分配替换字段的索引。

此外，还可以通过手动指定替换字段的索引来使其和参数匹配。

这三种方法可以混合使用。

<img src=image/2021-02-15_18-41.png>

注意到，Python函数在解析参数时，是按照顺序来的，必须先满足位置参数，才能考虑关键字参数。而且一旦分配索引，要将所有未命名的替换字段都分配，否则会报错。
<img src=image/2021-02-15_18-48.png>


你不仅可以提供值，而且可以提供变量名，进而访问它的组成部分。你还可以访问导入模块的方法，属性，变量和函数（`__name__`指定模块的名称）。
<img src=image/2021-02-15_18-56.png>


### 类型转换

**转换标志**允许你将替换后的值转换为不同格式和类型。

你可以使用`!s`，`!r`,`!a`字符串的表示形式：
- `!s`：普通的输出的字符串，与print得到的字符串相同
- `!r`：字符串字面量
- `!a`：ASCII码字符串

你还可以使用`:<typename>`将替换后的值转换为不同的类型：
| 类型      | 含义                                                                                               |
| --------- | -------------------------------------------------------------------------------------------------- |
| `b`       | 将整数转换为二进制数                                                                               |
| `o`       | 将整数转换为八进制数                                                                               |
| `x`或`X`s | 将整数转换为十六进制数(`X`相对于`x`用大写表示)                                                     |
| `c`       | 将整数解读为Unicode码点                                                                            |
| `d`       | 将整数视为十进制数进行处理                                                                         |
| `e`或`E`  | 使用科学计数法来表示小数(分别以`e`或`E`表示指数)                                                   |
| `f`或`F`  | 将小数表示为定点数(`F`对于`f`特殊值用大写表示)                                                     |
| `g`或`G`  | 自动在定点表示法和科学表示法之间选择，这是默认用于小数的说明符(`G`相对于`g`特殊值和指数用大写表示) |
| `s`       | 保持字符串的格式不变，这是默认用于字符串的说明符                                                   |
| `%`       | 将数表示为百分比值(乘以100)                                                                        |

### 宽度和精度

你可以设置字符串的格式，包括字符串的宽度，对齐方式（左对齐和右对齐），填充方式（用于填充的字符）。

你可以使用`:m.n<typename>`（其中m和n为整数）来指定格式字符串的宽度和精度，数默认右对齐，字符串默认左对齐，且都不进行任何形式的填充：

你还可以用`,`指出你要添加的千位分隔符：

### 对齐和填充

你可以使用`:<fill><align><sign>m.n<type>`（其中'<fill>'指定填充字符，<align>`指定填充方式，`<sign>`是符号说明符）：
- 对齐方式：
  - `^`：居中对齐
  - `<`：左对齐
  - `>`：右对齐
- 填充字符：
  - 是对对齐说明符的扩充，将剩余的字符用给定字符填充，而不是默认的空格
- 符号说明：
  - `0`：使用0来填充数字
  - `-`：默认负数前带有负号，而正数前不带符号
  - `+`：给正数前加上正号`+`
  - ` `：给正数前加上空格而不是`+`

## find

find的功能：在字符串中查找子串。如果找到，返回子串首字符的索引，否则返回-1。

当然，我们用成员资格运算也是相同的。

find的语法格式：
```python
s.find("<substring>", <begin>, <end>)
#指定子串，查找的起始索引和终止索引，起点和终点都是可选的
```
